# PLC点位表生成器

一款基于Python的PLC点位表、模块排序表和端子表生成工具，支持S7-1500和200SMART两种PLC类型。

## 功能特性

### 1. 点位表生成
- 从Excel模板读取设备定义
- 自动生成I/Q/IW/QW点位
- 支持自定义别名映射
- 支持关键词优先级排序

### 2. 模块排序生成
- I/Q模块：按偶数位、奇数位排序
- IW/QW模块：支持1500和200SMART两种模式
- 200SMART模式自动拆分+/-极性

### 3. 端子表生成
- 按设备类型自动分组
- KA继电器编号自动分配
- 支持优先级关键词排序
- 可选择是否添加设备间隔

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行

```bash
python plc_generator_ui.py
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `plc_generator_ui.py` | 主程序（整合版UI） |
| `generate_plc.py` | 点位表生成逻辑 |
| `generate_module.py` | 模块排序生成逻辑 |
| `generate_terminal.py` | 端子表生成逻辑 |

## 使用流程

1. **点位表生成**
   - 准备`点表模板.xlsx`模板文件
   - 设置起始地址
   - 点击"生成点位表"

2. **模块排序生成**
   - 设置I/Q/IW/QW模块数量和起始地址
   - 选择PLC类型（1500/200SMART）
   - 点击"生成排序表"

3. **端子表生成**
   - 选择PLC点位表文件
   - 选择PLC类型和24V标识
   - 点击"生成端子表"

## 优先级表

支持自定义关键词优先级，可导出/导入优先级表文件。关键词将自动用于设备名提取和排序。

## PLC类型说明

- **S7-1500**：IW/QW地址按标准排序
- **200SMART**：IW/QW地址按+/-极性分组，先全部+再全部-
